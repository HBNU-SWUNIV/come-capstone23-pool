# 📂 Package Description
## MnN
테이블의 다대다 매핑을 위한 패키지 입니다. <br>
멤버 테이블과 채팅 테이블, 멤버 테이블과 게시물 테이블은 각각 M : N(다대다) 매핑이 되어야하고 이를 위한 중간 테이블을 생성하였습니다.<br> 이 패키지에서는 이를 관리하는 기능을 수행합니다.
